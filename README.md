# 电力安全工器具制造厂WMS产品设计

## 项目简介
为苏州某电力安全工器具制造厂设计的WMS系统个人作品集，独立完成从需求调研到原型设计的全流程。

## 项目背景
- 该厂生产绝缘棒、接地线、绝缘手套等电力安全工器具
- 核心痛点：绝缘工器具需定期试验，超期使用存在安全风险；车间高频借还工器具，人工管理效率低

## 我的角色
产品经理（独立完成）

## 核心产出
- 需求分析与概念设计文档（含业务场景、用户角色、用户故事、AS-IS流程、TO-BE泳道图）
- 功能列表（7大模块，27个功能点）
- 产品架构图
- PC端原型（12个核心页面）
- PDA端原型（4个核心页面）

## 文件说明
- `需求文档.md` - 完整的需求分析与概念设计文档
- `prototypes/PC端/` - PC端所有原型页面截图
- `prototypes/PDA端/` - PDA端所有原型页面截图
- `prototypes/流程图/` - 业务流程图截图

## 如何查看
直接浏览本仓库的 `prototypes/` 文件夹即可查看所有原型页面截图。
